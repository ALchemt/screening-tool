# Agentic AI Developer @ ClauseAI

## About ClauseAI

ClauseAI builds autonomous AI agents for legal contract review, helping law firms and in-house legal teams analyze contracts 10x faster. We're a seed-stage startup backed by Bloomberg Beta and Khosla Ventures.

## Role

We're looking for a junior Agentic AI Developer to build multi-agent systems that autonomously review legal contracts, extract key clauses, and flag risks. You'll use LangGraph and the Model Context Protocol (MCP) to orchestrate agents that call specialized tools (PDF parsers, clause extractors, risk scorers) and collaborate to produce structured analysis reports. This is a greenfield opportunity to shape our agent architecture from the ground up.

## Responsibilities

- Design and implement multi-agent workflows using LangGraph for contract analysis pipelines
- Build tool-calling agents that interact with document parsers, vector databases, and external APIs
- Implement agent memory and state management for multi-turn contract review sessions
- Integrate Model Context Protocol (MCP) servers for standardized tool access across agents
- Develop evaluation scripts to measure agent accuracy, task completion rate, and reasoning quality
- Debug agent behavior, identify failure modes, and implement guardrails to prevent hallucinations
- Collaborate with legal experts to refine agent prompts and decision logic

## Required Qualifications

- 1-2 years experience with Python and LLM application development
- Hands-on experience with LangGraph or similar agent frameworks (CrewAI, AutoGen, or custom)
- Understanding of tool calling / function calling with OpenAI or Anthropic APIs
- Familiarity with state machines, graph-based workflows, and orchestration patterns
- Experience with prompt engineering and chain-of-thought reasoning techniques
- Comfortable reading and debugging complex Python codebases
- Strong written communication skills for documenting agent behavior
