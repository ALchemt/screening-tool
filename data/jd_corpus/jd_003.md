# AI Engineer (RAG) @ CartIQ

## About CartIQ

CartIQ builds AI-powered product discovery and customer support tools for Shopify Plus merchants. Our semantic search and automated support agents handle 500k+ queries per month across 80+ e-commerce stores.

## Role

We're hiring a junior AI Engineer focused on Retrieval-Augmented Generation to improve our product search and support chatbot accuracy. You'll build and refine document ingestion pipelines, experiment with chunking strategies, and tune retrieval performance using vector databases. You'll work closely with our ML lead to deploy RAG pipelines that answer customer questions using product catalogs, return policies, and order histories. This is a hands-on role with immediate production impact.

## Responsibilities

- Build document ingestion pipelines using LangChain or LlamaIndex for Shopify product data and support docs
- Implement and tune vector embeddings using OpenAI text-embedding-3 or Cohere Embed v3
- Set up and manage vector databases (Pinecone, Qdrant, or ChromaDB) for semantic search
- Experiment with chunking strategies (semantic, fixed-size, recursive) to optimize retrieval quality
- Develop RAG chains that combine retrieval with GPT-4 or Claude 3.5 for accurate answer generation
- Measure and improve retrieval metrics (MRR, NDCG) and end-to-end response quality
- Write unit tests and integration tests for ingestion and retrieval pipelines

## Required Qualifications

- 1-2 years experience with Python and familiarity with ML/AI concepts
- Hands-on experience with LangChain or LlamaIndex (personal projects or professional)
- Understanding of vector embeddings and similarity search (cosine, dot product)
- Experience with at least one vector database (Pinecone, Weaviate, Qdrant, ChromaDB)
- Ability to call and integrate OpenAI or Anthropic APIs in production code
- Comfortable with Git, virtual environments, and dependency management (Poetry, pip)
- Strong problem-solving skills and willingness to iterate based on metrics

## Nice-to-Have

- Experience with e-commerce platforms (Shopify, WooCommerce, BigCommerce)
- Familiarity with hybrid search (vector + keyword with BM25)
- Knowledge of prompt engineering and few-shot learning techniques
- Exposure to evaluation frameworks (RAGAS, TruLens, or similar)
- Understanding of data privacy and PII handling in customer data
