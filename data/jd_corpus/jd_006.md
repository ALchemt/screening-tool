# AI Integration Engineer @ PropelCRM

## About PropelCRM

PropelCRM is a real estate CRM and transaction management platform serving 5,000+ agents and brokerages. We're integrating AI to automate lead follow-up, listing descriptions, and contract summarization.

## Role

We need an AI Integration Engineer to embed OpenAI and Anthropic capabilities into our existing Ruby on Rails and React application. You'll design REST APIs and webhooks that connect our backend to LLM providers, handle async job processing for long-running AI tasks, and ensure reliable, low-latency responses. You'll work across the stack, from database schema changes to frontend UI for AI features, and collaborate closely with product to scope and ship new AI-powered workflows.

## Responsibilities

- Design and implement REST APIs for AI features (listing generation, email drafting, document summarization)
- Integrate OpenAI GPT-4 and Anthropic Claude APIs with robust error handling and retry logic
- Build webhook listeners and async job queues (Sidekiq) for background AI processing
- Implement rate limiting, cost tracking, and usage monitoring for LLM API calls
- Develop prompt templates and versioning system for consistent AI output across features
- Collaborate with frontend engineers to design intuitive UI for AI-generated content review and editing
- Write integration tests and monitor production performance with Datadog and Sentry

## Required Qualifications

- 3+ years backend engineering experience with REST API design and microservices
- Strong proficiency in at least one backend language (Ruby, Python, Node.js, or Go)
- Deep experience integrating third-party APIs with authentication, rate limiting, and error handling
- Hands-on experience with OpenAI and/or Anthropic APIs in production environments
- Proficiency with async job processing (Sidekiq, Celery, Bull, or similar)
- Solid understanding of webhooks, event-driven architecture, and idempotency
- Experience with relational databases (PostgreSQL, MySQL) and schema design

## Nice-to-Have

- Experience in real estate tech or CRM platforms (Salesforce, HubSpot, Zoho)
- Familiarity with Ruby on Rails or Django
- Knowledge of prompt engineering and LLM output validation techniques
- Experience with feature flags (LaunchDarkly, Flagsmith) for gradual AI rollouts
- Understanding of SOC 2 compliance and data residency requirements
