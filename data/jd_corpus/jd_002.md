# n8n Developer II @ Ledger Labs

## About Ledger Labs

Ledger Labs is a fintech infrastructure company providing real-time reconciliation and compliance automation for digital payment platforms. Our clients include neobanks, payment processors, and crypto on-ramps processing $2B+ in monthly transaction volume.

## Role

We need an experienced n8n Developer to own our self-hosted n8n instance and build custom nodes and workflows that connect our internal transaction processing system with customer CRMs (Salesforce, HubSpot) and ERPs (NetSuite, QuickBooks). You'll architect complex multi-step workflows involving data transformation, conditional logic, and external API calls. This role requires deep n8n expertise and the ability to write production-grade TypeScript for custom node development.

## Responsibilities

- Design, build, and maintain 20+ production n8n workflows handling financial data ingestion and reconciliation
- Develop custom n8n nodes in TypeScript for proprietary APIs and internal microservices
- Implement secure credential management and secret rotation for 15+ third-party integrations
- Optimize workflow performance and reduce execution time for high-volume data pipelines (10k+ records/day)
- Set up monitoring, alerting, and logging for workflow health using Prometheus and Grafana
- Collaborate with backend engineers to design webhook contracts and event schemas
- Lead code reviews for workflow changes and enforce version control best practices via Git

## Required Qualifications

- 3+ years building automation workflows, with 18+ months dedicated n8n experience
- Strong TypeScript/JavaScript skills and experience writing custom n8n nodes
- Deep understanding of n8n architecture, execution modes (main vs. worker), and scaling strategies
- Proficiency with self-hosted n8n deployment on Docker and Kubernetes
- Experience integrating with CRM (Salesforce, HubSpot) and ERP (NetSuite, QuickBooks) APIs
- Solid grasp of OAuth 2.0, API rate limiting, and error handling patterns
- Comfortable with PostgreSQL for n8n database management and workflow data storage

## Nice-to-Have

- Experience in fintech or payments infrastructure
- Familiarity with Redis for caching and queue management
- Knowledge of SOC 2 compliance and audit logging requirements
- Contributions to open-source n8n community nodes
- Experience with Temporal or Apache Airflow
